# React Hooks

This guide will walk you through the process of setting up and using the React library in your front-end project.

## Installation

::: code-group

```sh-vue [npm]
npm install @fuels/react --save
```

```sh-vue [pnpm]
pnpm add @fuels/react
```

```sh-vue [bun]
bun add @fuels/react
```

:::
